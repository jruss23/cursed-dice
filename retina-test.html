Approach Details:

  • A (Old): Game config has no resolution. Text uses resolution: DPR. Canvas buffer = CSS pixels.
  • B (Current): Game config has no resolution. Text uses resolution: DPR×2. High-res text resampled to 1x canvas.
  • C: Game config has resolution: DPR + roundPixels: true. Text uses resolution: DPR. Canvas buffer = device pixels.
  • D: Game config has resolution: DPR + roundPixels: false. Text uses resolution: DPR. Smooth anti-aliasing preserved.
  • E: Scale.NONE with width/height: ×DPR + zoom: 1/DPR. Canvas is native resolution, zoom scales display.
  • F (Proposed): Same as E but uses createText() helper with resolution: DPR and padding: {x:4, y:4}.
